# Contributing to Memoric

Thank you for your interest in contributing! Please follow the guidelines below to help us maintain a healthy project.

## Getting Started
- Fork the repository and create your branch from `main`.
- Ensure you have Python 3.9+.
- Create and activate a virtual environment.
- Install requirements: `pip install -r requirements.txt`.

## Development
- Run formatting and lint checks: `black . && flake8`.
- Run tests: `pytest`.
- Write clear commit messages and keep PRs focused.

## Code Style
- Use Black for formatting and Flake8 for linting.
- Prefer clear, descriptive names and early returns.

## Pull Requests
- Include tests for changes where applicable.
- Update documentation and examples if behavior changes.

## Reporting Issues
- Use the issue template, include steps to reproduce and environment details.
